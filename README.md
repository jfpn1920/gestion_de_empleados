## 👋 ¡Bienvenidos usuarios a mi proyecto! gestion de empleados

<p style="text-align: justify;">

Este proyecto consiste en un sistema desarrollado en Python que permite gestionar empleados como objetos dentro de una estructura organizada. Cada empleado es representado mediante una clase que contiene atributos fundamentales como nombre, cargo y salario, lo que permite modelar de manera clara la información de cada trabajador dentro del sistema.

El programa no solo almacena datos, sino que también permite realizar operaciones clave como el cálculo del total de salarios y la aplicación de aumentos porcentuales de forma automática. Esto simula un escenario real dentro de una empresa donde se requiere administrar información del personal y realizar ajustes salariales de manera eficiente.

Además, el proyecto está diseñado bajo los principios de Programación Orientada a Objetos (POO), promoviendo una estructura modular, reutilizable y escalable. La implementación de clases, métodos y bucles permite comprender cómo organizar mejor el código, facilitando su mantenimiento y futuras mejoras.

### 🧑‍💻 Lenguaje de programacion

#
### 🎯 Objetivos del proyecto
- Crear una clase `Empleado` con atributos básicos.
- Aplicar Programación Orientada a Objetos.
- Calcular el total de salarios de los empleados.
- Implementar aumentos salariales por porcentaje.
- Practicar el uso de listas y bucles.
- Organizar el código de forma estructurada y profesional.

#
### 🧠 Temas que se a aplicado
- Clases y Objetos  
- Método constructor `__init__`  
- Métodos personalizados  
- Encapsulamiento básico  
- Listas en Python  
- Bucles `for`  
- Cálculos matemáticos  
- Formateo de strings  
- Organización modular del código  

#
### ⚙️ Funcionamiento
1. Se crean empleados como objetos con nombre, cargo y salario.

2. Los empleados se almacenan en una lista dentro del sistema de gestión.

3. El sistema permite:
   - Mostrar información de cada empleado.
   - Calcular el total de salarios.
   - Aplicar aumentos salariales generales.

4. Se actualizan automáticamente los nuevos salarios después del aumento.
